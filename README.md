# Dead-Hardware-Checker
A github repo that lets you see your hardware info!


Hi this is Dead Hardware Checker!

This will check your hardware and give you 

The amount of storage you have across all your drives

Your windows version and product key

And full hardware report like cpu gpu memory etc!

To install install it from the releases

If you can't run it make sure you have Python installed (if you don't then go to the Microsoft store and type Python!)
